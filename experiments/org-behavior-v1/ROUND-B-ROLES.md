## 沟通规则

```
任何人 ←→ 任何人（全连接）
阶段间：全员同步脉冲
```

1. **任何龙虾可以随时 @任何其他龙虾**，没有信息路由限制
2. 任务分4个阶段，按顺序推进（需求理解 → 方案设计 → 技术实现 → 审查）
3. 每个阶段完成时，**全员发同步脉冲消息**，格式见任务书
4. 全员确认无分歧后才进入下一阶段
5. 阶段内鼓励跨角色插嘴
