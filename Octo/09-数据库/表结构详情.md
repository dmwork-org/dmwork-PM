# 表结构详情

[[概览|← 返回概览]]

所有 62 张表的详细字段定义，按模块分组。

> 完整 SQL 迁移文件见各模块 `modules/<module>/sql/` 目录。

---

## 用户系统（User 模块，15 表）

### user（用户主表）

| 字段 | 类型 | 约束 | 说明 |
|------|------|------|------|
| id | integer | PK AUTO_INCREMENT | 自增主键 |
| uid | VARCHAR(40) | NOT NULL UNIQUE | 用户唯一 ID |
| name | VARCHAR(100) | NOT NULL | 用户名字 |
| short_no | VARCHAR(40) | NOT NULL UNIQUE | 短编码 |
| short_status | smallint | NOT NULL DEFAULT 0 | 0=未修改，1=已修改 |
| sex | smallint | NOT NULL DEFAULT 0 | 0=女，1=男 |
| robot | smallint | NOT NULL DEFAULT 0 | **0=真实用户，1=机器人** |
| category | VARCHAR(40) | NOT NULL | 分类（service=客服, system=系统） |
| role | VARCHAR(40) | NOT NULL | 角色（admin, superAdmin） |
| username | VARCHAR(40) | NOT NULL | 登录用用户名 |
| password | VARCHAR(255) | NOT NULL | bcrypt 哈希（2026-02-28 扩展至255） |
| zone | VARCHAR(20) | NOT NULL | 手机区号 |
| phone | VARCHAR(100) | NOT NULL | 手机号 |
| chat_pwd | VARCHAR(40) | NOT NULL | 聊天密码 |
| lock_screen_pwd | varchar(40) | NOT NULL | 锁屏密码 |
| lock_after_minute | integer | NOT NULL DEFAULT 0 | 锁屏延迟（分钟，0=立即） |
| vercode | VARCHAR(100) | NOT NULL | 验证码（好友添加来源） |
| is_upload_avatar | smallint | NOT NULL DEFAULT 0 | 是否上传过头像 |
| qr_vercode | VARCHAR(100) | NOT NULL | 二维码验证码 |
| device_lock | smallint | NOT NULL DEFAULT 0 | 设备锁开关 |
| search_by_phone | smallint | NOT NULL DEFAULT 1 | 允许手机号搜索 |
| search_by_short | smallint | NOT NULL DEFAULT 1 | 允许短编号搜索 |
| new_msg_notice | smallint | NOT NULL DEFAULT 1 | 新消息通知 |
| msg_show_detail | smallint | NOT NULL DEFAULT 1 | 通知显示详情 |
| voice_on | smallint | NOT NULL DEFAULT 1 | 声音开关 |
| shock_on | smallint | NOT NULL DEFAULT 1 | 震动开关 |
| mute_of_app | smallint | NOT NULL DEFAULT 0 | App 禁音 |
| offline_protection | smallint | NOT NULL DEFAULT 0 | 离线保护 |
| version | bigint | NOT NULL DEFAULT 0 | 数据版本（乐观锁） |
| status | smallint | NOT NULL DEFAULT 1 | 0=禁用，1=可用 |
| bench_no | VARCHAR(40) | NOT NULL | 性能测试批次号 |
| app_id | VARCHAR(40) | NOT NULL FK | 所属 App |
| email | VARCHAR(100) | NOT NULL | 邮箱 |
| is_destroy | smallint | NOT NULL DEFAULT 0 | 是否已注销 |
| wx_openid | VARCHAR(100) | NOT NULL | 微信 openid |
| wx_unionid | VARCHAR(100) | NOT NULL | 微信 unionid |
| gitee_uid | VARCHAR(100) | NOT NULL | Gitee 用户 ID |
| github_uid | VARCHAR(100) | NOT NULL | GitHub 用户 ID |
| web3_public_key | VARCHAR(200) | NOT NULL | Web3 公钥 |
| msg_expire_second | bigint | NOT NULL DEFAULT 0 | 消息过期时长（秒） |
| created_at | timestamp | NOT NULL | 创建时间 |
| updated_at | timestamp | NOT NULL | 更新时间 |

**索引：** `UNIQUE INDEX uid`, `UNIQUE INDEX short_no_udx`

### user_setting（用户关系设置）

每对用户关系一条记录，涵盖好友关系、黑名单、个性化设置。

| 字段 | 类型 | 说明 |
|------|------|------|
| uid | VARCHAR(40) FK | 用户 |
| to_uid | VARCHAR(40) FK | 对方 |
| mute | smallint | 免打扰 |
| top | smallint | 置顶 |
| blacklist | smallint | 黑名单（1=已拉黑） |
| chat_pwd_on | smallint | 聊天密码 |
| screenshot | smallint DEFAULT 1 | 截屏通知 |
| revoke_remind | smallint DEFAULT 1 | 撤回通知 |
| receipt | smallint DEFAULT 1 | 消息回执 |
| remark | VARCHAR(100) | 好友备注 |
| flame | smallint | 阅后即焚开关 |
| flame_second | smallint | 阅后即焚销毁秒数 |
| version | BIGINT | 数据版本 |

**索引：** `UNIQUE INDEX to_uid_uid(uid, to_uid)`, `INDEX idx_user_setting_to_uid(to_uid)`

### device（用户设备）

| 字段 | 类型 | 说明 |
|------|------|------|
| uid | VARCHAR(40) FK | 所属用户 |
| device_id | VARCHAR(40) | 设备唯一 ID |
| device_name | VARCHAR(100) | 设备名称 |
| device_model | VARCHAR(100) | 设备型号 |

## 消息系统（Message/Webhook 模块，15+5 表）

### message（分片表结构，message~message4 共 5 张）

| 字段 | 类型 | 说明 |
|------|------|------|
| message_id | VARCHAR(20) | 消息唯一 ID（WuKongIM 分配） |
| message_seq | bigint | 频道内序列号（严格递增） |
| client_msg_no | VARCHAR(40) | 客户端消息编号（去重） |
| channel_id | VARCHAR(100) | 频道 ID |
| channel_type | smallint | 频道类型 |
| from_uid | VARCHAR(40) FK | 发送者 |
| type | integer | 消息类型（ContentType） |
| payload | blob | 消息体（加密二进制） |
| timestamp | integer | 发送时间戳（Unix 秒） |

详见 [[消息分片设计]]。

### message_extra（消息扩展，不分片）

| 字段 | 类型 | 说明 |
|------|------|------|
| message_id | VARCHAR(20) PK | 消息唯一 ID |
| revoke | smallint | 是否撤回 |
| revoker | VARCHAR(40) | 撤回者 |
| readed_count | integer | 已读数量 |
| content_edit | TEXT | 编辑后正文 |
| edited_at | integer | 编辑时间戳 |
| is_pinned | smallint | 是否置顶 |
| is_deleted | smallint | 是否删除 |
| version | bigint | 数据版本 |

### reminders（消息提醒）

| 字段 | 类型 | 说明 |
|------|------|------|
| channel_id | VARCHAR(100) | 频道 ID |
| channel_type | smallint | 频道类型 |
| reminder_type | integer | 1=@我，2=草稿 |
| uid | varchar(40) | 提醒的用户（空=全频道） |
| text | varchar(255) | 提醒内容 |
| message_id | VARCHAR(20) | 关联消息 |
| message_seq | bigint | 消息序列号 |
| version | bigint | 数据版本 |
| is_deleted | smallint | 是否已删除 |
| publisher | varchar(40) | 发布者 uid |

### conversation_extra（会话扩展）

| 字段 | 类型 | 说明 |
|------|------|------|
| uid | VARCHAR(40) | 用户 |
| channel_id | VARCHAR(100) | 频道 |
| channel_type | smallint | 频道类型 |
| browse_to | bigint | 预览到的最大 message_seq（影响未读数） |
| keep_message_seq | bigint | 保持的位置 |
| keep_offset_y | integer | 位置偏移量 |
| draft | varchar(1000) | 草稿 |
| version | bigint | 数据版本 |

## 群组系统（Group 模块，5 表）

### group（群组主表）

| 字段 | 类型 | 说明 |
|------|------|------|
| group_no | VARCHAR(40) PK | 群唯一编号 |
| name | VARCHAR(40) | 群名字 |
| creator | VARCHAR(40) FK | 创建者 uid |
| status | smallint | 0=不正常，1=正常 |
| forbidden | smallint | 全体禁言 |
| invite | smallint | 邀请开关 |
| allow_view_history_msg | smallint DEFAULT 1 | 新成员查看历史消息 |
| version | bigint | 数据版本 |
| notice | varchar(400) | 群公告 |
| avatar | VARCHAR(255) | 群头像 URL |
| group_type | smallint | 0=普通群，1=超大群 |
| allow_member_pinned_message | smallint | 允许成员置顶消息 |
| space_id | VARCHAR(40) FK | 所属 Space（2026-03-07 新增） |

### group_member（群成员）

| 字段 | 类型 | 说明 |
|------|------|------|
| group_no | VARCHAR(40) FK | 群唯一编号 |
| uid | VARCHAR(40) FK | 成员 uid |
| role | smallint | 0=普通，1=群主，2=管理员 |
| robot | smallint | 是否机器人 |
| invite_uid | VARCHAR(40) | 邀请人 uid |
| forbidden_expir_time | integer | 禁言时长（秒） |
| is_deleted | smallint | 是否已退出 |

**索引：** `UNIQUE INDEX group_no_uid(group_no, uid)`

### group_setting（用户群个性化设置）

| 字段 | 类型 | 说明 |
|------|------|------|
| uid | VARCHAR(40) FK | 用户 |
| group_no | VARCHAR(40) FK | 群 |
| mute | smallint | 免打扰 |
| top | smallint | 置顶 |
| show_nick | smallint | 显示昵称 |
| chat_pwd_on | smallint | 聊天密码 |
| flame | smallint | 阅后即焚 |
| flame_second | smallint | 销毁秒数 |
| version | BIGINT | 数据版本 |

## Bot 系统（Robot/BotFather/Base 模块）

详见 [[Bot数据模型]]。

### robot

| 字段 | 类型 | 说明 |
|------|------|------|
| robot_id | VARCHAR(40) UNIQUE | = user.uid |
| bot_token | VARCHAR(100) UNIQUE | bf_ 前缀，BotFather API 认证 |
| creator_uid | VARCHAR(40) FK | 创建者 |
| description | VARCHAR(500) | 机器人描述 |
| im_token_cache | VARCHAR(200) | 缓存的 IM Token |
| bot_commands | VARCHAR(1000) | 命令列表 JSON |
| auto_approve | tinyint DEFAULT 0 | 自动通过好友申请 |
| inline_on | smallint | 行内搜索开关 |
| status | smallint DEFAULT 1 | 启用状态 |
| app_id | VARCHAR(40) FK | 所属 App |

### app

| 字段 | 类型 | 说明 |
|------|------|------|
| app_id | VARCHAR(40) UNIQUE | App 唯一标识 |
| app_key | VARCHAR(40) | App 密钥（HMAC 验签） |
| app_name | VARCHAR(40) | App 名称 |
| app_logo | VARCHAR(400) | App Logo URL |
| status | integer | 0=禁用，1=可用 |

### user_api_key

| 字段 | 类型 | 说明 |
|------|------|------|
| uid | varchar(40) FK | 用户 |
| api_key | varchar(100) UNIQUE | API Key（uk_ 前缀） |
| space_id | varchar(40) FK | 绑定 Space |

**索引：** `UNIQUE KEY uk_uid_space(uid, space_id)`

## Space 系统（Space 模块，3 表）

### space

| 字段 | 类型 | 说明 |
|------|------|------|
| space_id | VARCHAR(40) UNIQUE | 空间唯一 ID |
| name | VARCHAR(100) | 空间名称 |
| description | VARCHAR(500) | 空间描述 |
| logo | VARCHAR(200) | Space Logo |
| creator | VARCHAR(40) FK | 创建者 uid |
| status | smallint DEFAULT 1 | 1=正常，0=已解散 |
| max_users | INT DEFAULT 0 | 最大成员数（0=不限） |
| preset_group_ids | TEXT | 预设群组 JSON 数组 |
| version | bigint | 数据版本 |

### space_member

| 字段 | 类型 | 说明 |
|------|------|------|
| space_id | VARCHAR(40) FK | 空间 ID |
| uid | VARCHAR(40) FK | 成员 uid（也可是 robot_id） |
| role | smallint | 0=成员，1=管理员，2=拥有者 |
| status | smallint DEFAULT 1 | 1=正常，0=已移除 |
| version | bigint | 数据版本 |

**索引：** `UNIQUE INDEX spacemember_spaceid_uid(space_id, uid)`

### space_invitation

| 字段 | 类型 | 说明 |
|------|------|------|
| invite_code | VARCHAR(40) UNIQUE | 邀请码 |
| space_id | VARCHAR(40) FK | 所属空间 |
| creator | VARCHAR(40) FK | 创建者 uid |
| max_uses | int DEFAULT 0 | 最大使用次数（0=不限） |
| used_count | int DEFAULT 0 | 已使用次数 |
| expires_at | timestamp NULL | 过期时间（NULL=永不） |
| status | smallint DEFAULT 1 | 1=有效，0=无效 |

| 字段 | 类型 | 说明 |
