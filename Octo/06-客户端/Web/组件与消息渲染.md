# Web 组件与消息渲染

> 30+ 核心 UI 组件 + 全消息类型渲染器（含 RTC 消息 9989-9994）。

[[概述|← Web 概述]] | [[Service层与Space|Service 层与 Space →]]

## 概述

`@dmwork/base` 包提供了完整的 UI 组件体系和消息渲染架构。消息渲染通过 `MessageManager` 注册表实现，支持任意 ContentType 的动态扩展。

## 组件层次结构

```
WKBase（全局弹窗宿主）
  └── WKLayout（三栏布局框架）
        ├── WKViewQueue（左侧导航 + 内容区路由队列）
        │     ├── WKNavHeader（导航头）
        │     └── WKViewQueueHeader（面板头）
        ├── ConversationList（会话列表）
        │     └── ListItemAvatar（头像列表项）
        ├── Conversation（聊天主区域）
        │     ├── MessageInput（消息输入框）
        │     │     ├── EmojiToolbar（Emoji 选择器）
        │     │     ├── ImageToolbar（图片上传）
        │     │     ├── FileToolbar（文件上传）
        │     │     └── SlashCommandMenu（/ 命令菜单）
        │     └── [消息列表]（react-virtuoso 虚拟滚动）
        │           └── MessageBase
        │                 └── [各 ContentType Cell]
        └── GlobalSearch（全局搜索）
```

## 消息渲染体系

### 渲染链路

```
WuKongIM JS SDK 接收消息
  │ WKSDK.register(contentType, () => new ContentClass())  // 解码
  ▼
WKApp.messageManager.registerCell(contentType, () => CellComponent)
  │
  ▼
Conversation 组件 → MessageWrap[]
  │
  ▼
<MessageBase>  // 气泡外框（头像/状态/右键菜单/多选）
  └── 具体 Cell（按 contentType 分发）
```

### MessageBase — 消息气泡基础框架

所有消息共享 `<MessageBase>` 外框，提供：
- **左右布局**：发送消息（右）/ 接收（左）
- **头像显示**：仅在序列末尾显示发送者头像
- **发送状态**：失败红色感叹号（点击重发）
- **多选框**：编辑模式下显示 Checkbox
- **右键菜单**：`ContextMenus`（复制/转发/回复/撤回等）
- **气泡尾巴**：SVG 三角形小尾巴

### MessageWrap — 消息包装器

| 字段 | 类型 | 说明 |
|------|------|------|
| `parts` | `Part[]` | 文本解析为多 Part（普通/mention/@/emoji/link） |
| `bubblePosition` | `BubblePosition` | first/middle/last/single，影响圆角样式 |
| `send` | `boolean` | 是否自己发送 |
| `fullStreamContent` | `string` | 流式消息完整内容 |
| `isStreaming` | `boolean` | 流式是否进行中（显示光标动画） |
| `flame` | `boolean` | 阅后即焚标识 |

## 各消息 Cell 实现详解

### TextCell（文本消息）

```typescript
// Part 类型：普通文本 / @提及 / Emoji / 链接
type PartType = 'text' | 'mention' | 'emoji' | 'link'

// 特性
- 流式消息：实时追加，末尾闪烁光标（wk-stream-cursor CSS 动画）
- 引用回复：展示被引用消息发送者头像 + 名字 + 摘要，点击定位
- 消息编辑：显示编辑后内容（remoteExtra.isEdit = true）
- 安全 URL：链接渲染前验证安全性（防 XSS）
```

### ImageCell（图片消息）

```typescript
- 全屏预览：react-viewer
- 按比例缩放：imageScale()，最大 200×200
- 支持 caption（图片说明）
- 支持 mentionUids（图片 @ 提及）
```

### VoiceCell（语音消息）

```typescript
- 格式：AMR（benz-amr-recorder 播放）
- WaveCanvas：波形图 Canvas 可视化
- 已读/未读：红点标识
- 点击播放：AMR 解码并播放
```

### FileCell（文件消息）

```typescript
// 按扩展名显示彩色图标
PDF → 红色    DOC/DOCX → 蓝色
XLS/XLSX → 绿色  PPT/PPTX → 橙色
其他 → 默认灰色

// 文件大小格式化：B / KB / MB / GB
// 下载：WKApp.dataSource.commonDataSource.getFileURL()
```

## 消息注册机制

```typescript
// BaseModule.init() 注册所有消息类型
class BaseModule implements IModule {
    init() {
        WKApp.messageManager.registerCell(ContentType.TEXT,
            () => TextCell)
        WKApp.messageManager.registerCell(ContentType.IMAGE,
            () => ImageCell)
        // ... 注册所有类型
        
        // 注册 WuKongIM SDK 解码器
        WKSDK.shared().register(ContentType.TEXT,
            () => new TextContent())
        // ...
    }
}
```
