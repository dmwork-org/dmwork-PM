# DMWork Android 客户端概述

> Java/Kotlin + Gradle 模块化 Android IM 客户端，6 厂商推送全覆盖。

[[概述|← 客户端概述]] | [[推送系统|Android 推送系统 →]]

## 概述

`dmwork-android` 是 DMWork 的 Android 客户端，基于 WuKongIM Android SDK，采用 Gradle 多模块架构。

**关键标识**：
- **包名**：`com.dmwork.im`
- **rootProject.name**：`TsddClient`（遗留历史品牌名，来自 TangSengDaoDao 时期）
- **语言**：Java（主要）+ Kotlin（`app` 模块 Activity）
- **多环境**：dev（测试服）/ prod（生产服）productFlavors

> ⚠️ **CORRECTED**：`rootProject.name=TsddClient` 是从上游 fork 保留的遗留标识，实际产品已是 DMWork。

## Gradle 模块结构

```
dmwork-android/
│
├── app/                  ← 主应用模块（入口 + 组装）
├── wkbase/               ← 基础模块（工具/网络/DB/UI组件）
├── wkuikit/              ← 聊天 UI 核心（消息/会话/群组/联系人）
├── wklogin/              ← 登录注册模块
├── wkpush/               ← 多厂商推送模块
├── wkscan/               ← 扫一扫二维码模块
└── MyLibs/               ← 厂商推送 AAR（本地依赖）
    ├── oppopush/         ← OPPO push-3.0.0.aar
    ├── vivopush/         ← VIVO push AAR
    └── rlottie/          ← Lottie 动画库（CORRECTED: 补充此模块）
```

> ⚠️ **CORRECTED**：`MyLibs:rlottie` 模块是通过源码验证补充的，原始分析文档中未记录。rlottie 用于播放 Lottie 动画（表情/互动效果）。

### 6 个 Gradle 模块

| 模块 | Package | 职责 |
|------|---------|------|
| `app` | `com.dmwork.im` | 入口、Application 初始化、主 Activity |
| `wkbase` | `com.chat.base` | 基础工具、网络层、本地 DB、UI 组件库 |
| `wkuikit` | `com.chat.uikit` | 聊天 UI（消息气泡/会话/群组/联系人） |
| `wklogin` | `com.chat.login` | 登录/注册/OAuth/扫码授权 |
| `wkpush` | `com.chat.push` | 6 厂商推送初始化与接收 |
| `wkscan` | `com.chat.scan` | 摄像头扫码（好友/群/PC登录） |

### 依赖方向（单向）

```
app → wkuikit / wklogin / wkpush / wkscan → wkbase
                                               ↓
                                        WuKongIMAndroidSDK
```

### app — 主应用模块

**`TSApplication.kt`**（初始化顺序）：

```kotlin
fun initAll() {
    WKMultiLanguageUtil.init()     // 1. 多语言
    WKBaseApplication.init()       // 2. 基础初始化（网络/DB）
    Theme.applyTheme()             // 3. 主题
    initApi()                      // 4. API URL（BuildConfig/SharedPrefs）
    WKLoginApplication.init()      // 5. 登录模块
    WKScanApplication.init()       // 6. 扫码模块
    WKUIKitApplication.init()      // 7. 聊天 UI 模块
    WKPushApplication.init()       // 8. 推送模块
}
```

## EndpointManager — 端点总线

Android 端 `EndpointManager` 是跨模块通信核心，同 Web/iOS 端设计理念：

```java
// 注册
EndpointManager.getInstance().setMethod("endpoint_name", args -> {
    // 处理逻辑
    return result;
});

// 调用
EndpointManager.getInstance().invoke("endpoint_name", args);
```

关键端点：
- `"show_tab_home"` → 登录成功后跳转主界面（`TabActivity`）
- `"main_show_home_view"` → 登录模块调用，触发主界面展示

## WuKongIM SDK 集成

```groovy
// wkbase/build.gradle
api 'com.github.WuKongIM:WuKongIMAndroidSDK:1.4.7'
// via JitPack: https://jitpack.io
```

SDK 封装：
- TCP/WebSocket 长连接管理
- 消息发送/接收/持久化
- 会话列表管理
- 消息已读回执
- 频道（Channel）信息管理
- 端到端加密（Signal 协议）

前后台切换策略：
- App 进入前台 → 连接 WuKongIM
- App 进入后台且无通话 → 断开（节省电量 + 推送接管）

## UI 架构流程

```
TSApplication（Application）
└── SpaceGuideActivity（选择服务器地址）
      └── LoginActivity（登录/注册）
            ↓ 登录成功
          EndpointManager.invoke("show_tab_home")
            ↓
          TabActivity（主 Tab）
          ├── Tab1: ChatFragment（消息列表/最近会话）
          │         └── ChatActivity（聊天页）
          ├── Tab2: 联系人 Fragment/Activity
          └── Tab3: 我的 Fragment/Activity
```
